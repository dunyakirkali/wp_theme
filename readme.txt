## Guide

### Define the left image

The image that's displayed on the left column of the theme is fetched from the current page's featured image.

### Define icons for pages on mobile-menu

You can add a custom field to any page, called 'Page Icon'. This icon will be used to generate the navigation menu on small screens. If no icon is specified then a **default icon** (fa-bars) will be used.
